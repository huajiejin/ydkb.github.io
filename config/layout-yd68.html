刷机方式：
首先重点，这不是gh60，不要用zadig去安装驱动，这个刷固件是免驱的。
固件功能正常时，插入数据线同时按住左上角ESC键，键盘进入刷机模式。这时将hex拖到ydkbs-relflash.exe，固件将自动开始刷新。
如果刷错了别的固件，造成固件功能不正常了，先关掉电源开关，再执行上面的操作。

功能补充说明:
蓝牙模式下不支持NumLock、CapsLock、ScrollLock指示灯。但是caps灯设为capslock时，可指示大小写，只是不与系统同步（实际就是每次按Capslock时灯切换亮灭）。在不同步的时候，可用Shift+Capslock，来手动让capslock指示灯同步。
使用tada68塑料壳的，最上面6颗RGB底灯看不见的，可以在LEDMAP里将“RGB上6颗”设置为禁用，这样它们就不会亮了。

Reflash Firmware:
YD68 uses BootHID bootloader. So it's no need to install any driver manually for it.
Disconnect usb cable from the keyboard first. Drag .hex file to ydkb-relflash.exe, as it says next you need get your device into bootloader.
Now, insert the usb cable to your keyboard and meanwhile keep holding Esc key. When it begins flashing, Esc can be released.
If the firmware is wrong, turn off the battery switch before doing above.

More to say:
1. Under bluetooth mode, Numlock, Capslock and Scroll lock LEDs are not supported. But when CAPS in LEDMAP is set to CAPS LOCK, this led can be an indicator for Capslock. When Capslock is pressed, it changes this led. If it is not synchronous with system, press Shift+Capslock which changes capslock state but doesn't change the led.
2. Setting RGB上6颗 to Disable can disable them as in tada68's plastic case, they can not be seen.
